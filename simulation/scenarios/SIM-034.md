# Localization — local gearbox assembly  (SIM-034)

> A fully-imported gearbox is expensive; evaluate local assembly from mixed sourcing.

## Context
Imported complete gearbox (high cost, long lead). Option: localize assembly in Thailand,
sourcing gears locally + import only specialized internals.

## Data Given
- Imported complete: high price + 10-wk lead + FX.
- Local assembly: needs qualified gear suppliers + assembly capability + IP/know-how.

## The Ask
Assess make/localize vs continue importing.

## Expected Pipeline
- [make-vs-buy](../../brain/make-vs-buy.md), [cost-down-framework](../../brain/cost-down-framework.md), [supplier-risk-framework](../../brain/supplier-risk-framework.md); multi-supplier selection + audit; [nda](../../workflows/nda.md) for know-how.

## Gold-Standard Approach
TCO over import (price, FX, lead, inventory) vs localized (component sourcing + assembly +
qualification + risk). Phase it: localize low-risk components first; keep specialized
internals imported until local source qualified. NDA before sharing gearbox design. Big
strategic move = Human Owner (D3/D4); strong continuity + FX benefit if executed.

## Pitfalls
- ❌ Localizing everything at once (quality/continuity risk).
- ❌ Sharing gearbox IP without NDA.
- ❌ Ignoring assembly capability/qualification cost.

## Scoring Rubric (0–5 each)
- [ ] TCO import vs localized · [ ] Phased de-risking · [ ] NDA before IP shared
- [ ] Suppliers qualified · [ ] Strategic decision escalated · [ ] Lesson captured

## Lessons to Capture
- Phased localization; protect IP; FX + lead as TCO drivers.
