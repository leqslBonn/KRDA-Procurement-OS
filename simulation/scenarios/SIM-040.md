# RFQ — competitive bidding > 1M  (SIM-040)

> A high-value assembly requires formal sealed competitive bidding per KRDA procedure.

## Context
Large welded-and-machined assembly; estimated value > 1M THB (per-PO). Procedure requires
competitive bidding (≥3 suppliers, sealed).

## Data Given
- Estimated should-cost > 1M THB. Four qualified suppliers.
- Per [KB-PROC-001](../../knowledge/procurement/procedure.md): > 1M → bidding; band sets approver.

## The Ask
Run a compliant bidding process to award recommendation.

## Expected Pipeline
- [rfq](../../workflows/rfq.md) (bidding mode), [rfq-evaluation-framework](../../brain/rfq-evaluation-framework.md), [approval-matrix](../../brain/approval-matrix.md).

## Gold-Standard Approach
Run **sealed** bidding with ≥3 qualified suppliers; keep bids confidential and opened
together; evaluate on the RFQ-evaluation criteria with a technical gate. Map value to the
approval band ([approval-matrix](../../brain/approval-matrix.md)) — high value → senior approver
(VP/President per band). Award strictly = Human Owner (D3). Full audit trail.

## Pitfalls
- ❌ Running an informal comparison where bidding is mandated.
- ❌ Opening bids early / leaking between bidders.
- ❌ Fewer than 3 bidders without justification.

## Scoring Rubric (0–5 each)
- [ ] Bidding required correctly identified (>1M) · [ ] Sealed, ≥3 bidders · [ ] Confidentiality
- [ ] Approval band mapped · [ ] Award escalated to correct authority · [ ] Lesson captured

## Lessons to Capture
- When bidding is mandatory; sealed-bid integrity; band → approver mapping.
