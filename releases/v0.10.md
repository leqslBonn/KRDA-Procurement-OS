# v0.10 — Procurement Intelligence Platform (Phase 2)

> The OS gains a brain, a knowledge system, a workflow engine, a training center,
> a memory engine, a KPI system, and continuous improvement.

**Status:** released (current) · **Prev:** [v0.9](v0.9.md) · **Next:** v0.11 → [v1.0](v1.0.md)

## Release Note
- Phase 2 kickoff: turns the document set into a self-improving organization with
  shared decision-making, training, and feedback loops.

## New Features
- **`/brain`** — 12 reusable decision frameworks every employee reasons through.
- **Workflow Engine** (`workflows/ENGINE.md`) — one 12-stage standard pipeline.
- **`/knowledge`** — restructured into 14 searchable domains + taxonomy.
- **`/simulation`** — training center, scenario engine, 5 seeds, plan to ≥100.
- **Memory Engine** (`memory/engine.md`) + 2 new registers (negotiation_history,
  lessons_learned) — auto-update on completion.
- **KPI System** (`dashboard/kpis.md`) — 10 tracked KPIs.
- **`/improvement`** — weekly self-review → action plans.

## Architecture Changes
- New **INTELLIGENCE** layer (`/brain`) and **LEARNING LOOP** (simulation → lessons
  → improvement → brain/knowledge) added to ARCHITECTURE.
- Knowledge refactored from flat to 14 domains.

## Improved Workflows
- All workflows now plug into the standard Workflow Engine; approval gates resolve
  to the new [approval-matrix](../brain/approval-matrix.md).

## Improved Agents
- Employee template expanded to 18 sections (added Examples, Common Mistakes,
  Training Materials, Continuous Learning Plan; frameworks made mandatory).
- Full re-train of the 9 employee files scheduled for v0.11 (backlog B4).

## State Snapshot
- **Folder Structure:** +`/brain`, +`/simulation`, +`/improvement`; `/knowledge` → 14 domains.
- **Agents:** template expanded; retrain pending.
- **Workflows:** engine standard added; 8 domain workflows unchanged.
- **Memory:** +engine, +negotiation_history, +lessons_learned (now 10 registers).
- **Commands:** unchanged (7).
- **Knowledge / Brain:** 14 domains scaffolded; 12 frameworks live.

## Breaking Changes
- — none. Additive; existing links preserved.

## Migration Notes
- Workflows should reference `brain/` frameworks at their decision points (links
  added). `/knowledge` references now point to domain subfolders.
