# KRDA Spend Category Taxonomy

> Canonical category tree for spend analysis, commodity database, and sourcing
> strategy. All `CMD-` codes originate here. Link here; never copy.
>
> KRDA scope: R&D agricultural machinery for ASEAN (rice, sugarcane, cassava, maize).
> Two procurement streams: **Technical** (R&D parts/materials) and **General** (company operations).

## Coding Rules

1. Every supplier in `supplier_database.md` maps to ≥1 CMD code.
2. Every RFQ references a CMD code in the RFQ record header.
3. Every cost-down case references the CMD code of the part.
4. New sub-category: add a row here + add to `commodity_database.md` index.
5. CMD codes are **permanent** — never delete; set to `inactive` if no longer used.
