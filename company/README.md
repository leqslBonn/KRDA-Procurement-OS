# /company

Context and **governance** for the KRDA AI Procurement Department: who we are,
why we exist, how we decide, and how we are organized. This is the onboarding
manual every agent reads — and the constitutional layer they are bound by.

> บริบทและธรรมนูญของแผนกจัดซื้อ AI ของ KRDA — คู่มือปฐมนิเทศที่ทุก agent ต้องอ่าน
> และเป็นชั้นกฎสูงสุดที่ทุก agent ผูกพัน

## Conventions

- Company **facts** live here and are referenced elsewhere — never duplicated.
- Keep entries factual and dated. Mark unknowns as `TBD`.
- This folder describes *context*, not *operations*. Operational records live in
  `/projects`, `/rfq`, `/cost_down`, and `/contracts`.
