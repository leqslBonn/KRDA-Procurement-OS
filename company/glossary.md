# Procurement Glossary (EN/TH)

Shared vocabulary for the whole system. Define a term **once here**; reference it
elsewhere. Add terms as the system grows.

| Term | EN meaning | TH note |
|------|-----------|---------|
| RFQ | Request for Quotation | คำขอใบเสนอราคา |
| RFI | Request for Information | คำขอข้อมูลผู้ขาย |
| PR | Purchase Requisition | ใบขอซื้อ |
| PO | Purchase Order | ใบสั่งซื้อ |
| BOM | Bill of Materials | รายการวัสดุ |
| MOQ | Minimum Order Quantity | จำนวนสั่งซื้อขั้นต่ำ |
| Lead Time | Time from order to delivery | ระยะเวลาส่งมอบ |
| Should-Cost | Modeled "fair" cost of a part | ต้นทุนที่ควรจะเป็น |
| VA/VE | Value Analysis / Value Engineering | วิเคราะห์/ปรับปรุงคุณค่าเพื่อลดต้นทุน |
| TCO | Total Cost of Ownership | ต้นทุนรวมตลอดอายุการใช้งาน |
| Cost-Down | Cost-reduction initiative | กิจกรรมลดต้นทุน |
| Bid Tab | Bid tabulation / quote comparison | ตารางเปรียบเทียบราคา |
| Supplier Qualification | Approving a supplier to be used | การรับรองผู้ขาย |
| Single / Sole Source | One supplier by choice / by necessity | แหล่งเดียวโดยเลือก/โดยจำเป็น |
| SAP | ERP system used for purchasing | ระบบ ERP สำหรับจัดซื้อ |
| KPI | Key Performance Indicator | ตัวชี้วัดผลงาน |
