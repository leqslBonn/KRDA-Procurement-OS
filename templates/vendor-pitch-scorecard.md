## Gate Check (any ✗ → exclude or POC-only)

- [ ] KRDA owns its farm/operational **data** (not the vendor)
- [ ] Security/privacy adequate (auth, encryption, hosting location)
- [ ] Working product exists for a production decision (else → POC only)
- [ ] Startup has runway / continuity plan / **source-code escrow** offered

## 🚩 Red Flags (note any)

- [ ] All slides, no live demo / no real customer
- [ ] Evasive on data ownership or security
- [ ] One-person dependency; thin funding
- [ ] Proprietary lock-in, no data export
- [ ] Price-only pitch; vague on integration/support
