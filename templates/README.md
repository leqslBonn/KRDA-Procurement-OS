# /templates

Standard document formats used across the system — the "forms" of the
procurement department. Fill a copy; never redesign per use.

> เทมเพลตเอกสารมาตรฐาน — กรอกสำเนา ไม่ต้องออกแบบใหม่ทุกครั้ง

## One Standard, Every Agent

**Every employee uses the same template for the same document type — no private
formats.** This is what makes the company systematic: one form per artifact, owned
here, improved here. Fill a copy into the owning record folder; never redesign per
use. Templates are referenced by agents, skills, and workflows — keep IDs stable.

> หลักสำคัญ: หนึ่งเอกสาร = หนึ่ง template ที่ทุก agent ใช้ร่วมกัน → บริษัทเป็นระบบ
